# Epic 001 — Identity & Application Shell

## Problem
FacilityOS needs a professional shell and identity system before business modules are added.

## Goal
Create the permanent SaaS foundation: login, account menu, portals, navigation, Mission Control, and multi-tenant company setup.

## Acceptance Criteria
- App runs in Codespaces.
- Supabase connects through `.env`.
- Login modal opens from logo.
- Account panel shows current session.
- Company can be created and linked to profile.
- Navigation changes by portal.
- GitHub Pages workflow exists.
